**3 файла:**
*point.h* - заголовочный файл, содержит описание структур и функций
*functions.c* - дополнительный файл, содержащий функции
*main.c* - основной файл, содержит основной код

**Описание функций**
*createPoint* - создаем точку через указатели
*printPoint* - выводит точку через printf
*calculateDistance* - считает Δx, Δy, Δz и находит расстояние между точками
*freePoint* - освобождает память

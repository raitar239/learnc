Employee - структура сотрудника (name, id, level)

**Оценка 3:**\
read_csv - читаем CSV-файл:\
    - имя файла передаётся через аргументы командной строки `argv[1]`\
    - построчное чтение через `fgets()`, разбор полей через `strtok()` по `';'`\
    - обработка ошибок открытия файла и битых строк\
print_table - выводим таблицу всех сотрудников в консоль

**Оценка 4:**\
save_database - сохраняем массив структур в бинарный файл `database.dat` через `fwrite()`\
search_by_id - открываем `database.dat`, читаем через `fread()`, запрашиваем ID у пользователя:\
    - найден — выводим данные сотрудника\
    - не найден — выводим ошибку

**Оценка 5:**\
compress_rle - сжимаем `database.dat` в `database.rle`:\
    - находим последовательности одинаковых байт\
    - записываем пары `[count][value]`\
decompress_rle - восстанавливаем исходный файл из `.rle`\
print_compression_report - выводим отчёт о сжатии:\
    - размер до / после в байтах и Кб\
    - процент сжатия\
    - проверка совпадения размеров оригинала и восстановленного файла

**★ системные вызовы:**\
read_csv переписан на *низкоуровневый* I/O:\
    - `open()` вместо `fopen()`\
    - `read()` вместо `fgets()`, строки накапливаем вручную побайтово\
    - `strchr()` вместо `strtok()` для разбора по `';'`\
    - `close()` вместо `fclose()`\
    - `write(STDOUT_FILENO)` / `write(STDERR_FILENO)` вместо `printf()` / `fprintf()`

**Как запустить**:\
```bash
gcc -o main main.c
./main staff.csv
```

**ЛАБОРАТОРНАЯ РАБОТА 13**
LINKED LISTS
-
room - структура комнаты\
Node - структура узла (3 указателя на узел)

*create_random_room - создаем комнату\
*create_node - создаем узел\
*build_row - создаем ряд (если doubly=1, то двусвязный, иначе односвязный)\
connect_rows - связываем строки (top & btm)\
free_row - освобождаем память

draw_room - рисуем комнату:\
    - масштабируем размеры под консоль\
    - делаем границы\
    - стрелка указывает в какую комнату можно перейти\
    - делаем надпись в центре

navigation - само управление:\
    - выводим подсказку по управлению\
    - считываем что символы которые вводит пользователь\
    - переходим соответсвенно введенной команде\

Использовались:\
`snprintf` - удобного создания переменных\
`do { cmd = getchar();} ...` - чтобы постоянно не вводилось '\n' и ''
